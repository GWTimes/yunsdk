# 应用推送开发指南

## 1. 概览

包含了离线推送token上报，在线消息回调（包括设备消息、系统消息）

### 1.1 模块名

PushMgr

## 2. SDK接口

#### 2.1 注册离线消息

###### 定义描述
| 名称  | 版本  | 描述     |
| ----- | ----- | -------- |
| registerRemoteNotification | 1.0.0 | 注册离线消息 |

###### 请求参数
| 参数 | 类型 | 非空 | 描述 |
| ---- | ---- | ---- | ---- |
| deviceToken | String | 是 | 离线推送deviceToken |
| responseHandler | Object | 是 | 注册结果回调 |

###### 返回参数
| 类型 | 非空 | 描述     |
| ---- | ---- | -------- |
|      |      | 无返回值 |
